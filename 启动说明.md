# 🔥 阅后即焚项目启动说明

## 环境要求

在运行此项目前，请确保您的系统已安装以下软件：

### 必需软件
1. **Node.js 16+** - [下载地址](https://nodejs.org/)
2. **npm** - 通常随Node.js一起安装

### 验证安装
打开命令行工具，运行以下命令检查是否已安装：

```bash
node --version  # 应该显示版本号如 v18.17.0
npm --version   # 应该显示版本号如 9.6.7
```

## 安装步骤

### 1. 安装后端依赖
```bash
cd backend
npm install
```

### 2. 安装前端依赖
```bash
cd ../frontend
npm install
```

### 3. 安装根目录依赖（开发工具）
```bash
cd ..
npm install
```

## 运行项目

### 开发模式（推荐）
在项目根目录运行：
```bash
npm run dev
```

这将同时启动：
- **后端服务**: http://localhost:3000
- **前端开发服务器**: http://localhost:5173

### 单独启动

#### 启动后端服务
```bash
cd backend
npm run dev
```

#### 启动前端服务
```bash
cd frontend
npm run dev
```

## 访问应用

1. 打开浏览器访问: http://localhost:5173
2. 创建阅后即焚消息
3. 复制生成的链接发送给朋友
4. 朋友打开链接后，消息将在10秒后自动销毁

## 常见问题

### Q: 无法安装依赖包
A: 检查网络连接，或尝试使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
```

### Q: 端口被占用
A: 修改backend/server.js中的PORT变量，或杀死占用端口的进程

### Q: 数据库文件不存在
A: 首次运行时会自动创建SQLite数据库文件
